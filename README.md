# GradPath

GradPath is an academic internship and placement tracking system developed as a semester project.

## Features
- Student, company, and internship management
- JDBC-based backend connectivity
- MySQL database integration
- Java Swing GUI
- Activity logging and notifications

## Tech Stack
- Java
- JDBC
- MySQL
- Java Swing

## Project Scope
This project focuses on backend logic, database design, and integration rather than production-level deployment.
